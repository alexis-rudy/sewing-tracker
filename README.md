# Sewing Tracker

A web application to create, track, and manage sewing projects with authentication, CRUD, tagging, and progress updates.

## Features
- User authentication (sign up, sign in, password reset)
- Project CRUD (create, read, update, delete with soft delete)
- Progress tracking timeline per project
- Tagging and filtering
- Search and sorting
